# Database E_commerce (SQL)

## Overview

This project analyzes customer orders and payments using SQL JOINs.

##  Tables Used

* Customers
* Orders
* Payments

## Key SQL Concepts
* INNER JOIN
* LEFT JOIN
* GROUP BY
* HAVING
* NULL Handling
##  Tasks Covered

1. Customer Orders
2. All Customers
3. Invalid Orders
4. Order Payment Status
5. Customers Without Orders
6. Orders Without Payment
7. Total Spending
8. Fully Paid Customers
9. Highest Order Per Customer
10. Top Customers
