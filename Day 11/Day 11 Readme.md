# Insurance Claim & Fraud Analysis Dashboard

##  Project Overview

This project focuses on analyzing insurance claim data to identify trends, detect fraud patterns, and provide actionable business insights using Power BI. The dashboard helps stakeholders understand claim behavior across different customer segments, locations, and incident types.

---

##  Objectives

* Analyze claim trends over time
* Identify high-risk regions and customer segments
* Detect fraudulent claims and patterns
* Understand relationship between premium and claim amounts
* Build an interactive dashboard for decision-making

---

##  Dataset Details

The dataset includes:

* Policy details (policy number, premium, deductible)
* Customer demographics (age, occupation, education)
* Incident details (date, type, severity, location)
* Claim details (total claim, injury/property/vehicle claims)
* Fraud reported status

---

##  Data Cleaning Steps

* Removed null and duplicate values
* Standardized column names
* Converted data types (Date, Numeric, Text)
* Created new columns:

  * Age Group
  * Claim Status (Fraud / Genuine)
  * Month & Year

---

##  Key KPIs

* Total Claims
* Total Claim Amount
* Fraud Count
* Fraud Rate (%)

---

## Dashboard Features

*  Claim Trends (Monthly Analysis)
*  Claims by State (Geographic Analysis)
*  Incident Type Analysis
*  Fraud vs Genuine Claims Distribution
*  Customer Segmentation (Age Group)
*  Premium vs Claim Amount (Scatter Analysis)


##  Interactivity

The dashboard includes slicers for:

* Year
* State
* Incident Type
* Age Group
* Fraud Status

##  Key Insights

* Certain states show higher claim and fraud rates
* Middle-age customers (26–40) contribute to higher claims
* Specific incident types have more fraud cases
* Higher premium policies often have higher claim amounts

##  Recommendations

* Implement stricter fraud detection in high-risk areas
* Improve claim verification process
* Monitor high-value claims closely
* Optimize premium pricing strategy

---

##  Tools Used

* Power BI
* Power Query
* DAX

---

##  Conclusion

This dashboard provides a comprehensive view of insurance claims and fraud trends, helping businesses make data-driven decisions and reduce financial risks.
