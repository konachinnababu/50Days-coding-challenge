## Project Overview

This project analyzes IT help desk ticket data to identify patterns, improve support efficiency, and provide actionable insights for better decision-making. The analysis focuses on ticket volume trends, resolution times, common issues, and analyst performance.

---

##  Objectives

* Analyze ticket volume trends over time
* Measure average resolution time
* Identify most common issues (root causes)
* Evaluate analyst performance

---

##  Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Dashboard Design

---

##  Dataset Details

The dataset contains 150 IT help desk tickets with the following fields:

* Ticket ID
* Submission Date
* Resolution Date
* Category (Network, Software, Hardware, Access)
* Assigned Analyst
* Description
* Priority


##  Key Features

###  Summary Dashboard

* Total Tickets KPI
* Average Resolution Time KPI
* Top Issue Category
* Best Performing Analyst
* Interactive charts with slicers

###  Data & Calculations

* Resolution Time (in hours)
* Day of Week
* Month
* Hour

###  Detailed Analysis
* Ticket trends by day/month
* Resolution time by category and priority
* Analyst performance comparison
* Keyword-based issue analysis

### Recommendations

* Implement self-service tools for common issues
* Improve network infrastructure
* Provide training for slower analysts
* Optimize staffing during peak hours

## Key Insights

* High volume of tickets observed during peak working hours
* Software and access-related issues are most frequent
* Significant variation in analyst performance
* Recurring issues indicate opportunities for automation

## Outcome

This analysis helps the IT department:

* Reduce ticket resolution time
* Improve team efficiency
* Proactively fix recurring issues
* Enhance overall IT support performance
