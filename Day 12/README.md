# 📊 MGNREGS Performance Analysis Dashboard

## 📌 Overview
This Power BI dashboard provides a comprehensive analysis of the **MGNREGS (Mahatma Gandhi National Rural Employment Guarantee Scheme)** performance across different states and years. It highlights key metrics such as demand vs. provided work, expenditure trends, and work completion efficiency.

---

## 🎯 Objectives
- Analyze household demand vs. work provided
- Track government expenditure trends
- Evaluate state-wise performance
- Measure work completion efficiency
- Identify gaps in implementation

---

## 📈 Key KPIs
- 44M – Total Household Demand  
- 38M – Total Household Provided  
- 122K Cr – Total Expenditure  
- 87.79% – Work Completion Rate  

---

## 📊 Dashboard Features

### 1. Expenditure Trend Analysis
- Line chart showing Expenditure per Person Day over years (2017–2021)
- Helps identify growth patterns and spending efficiency

### 2. State-wise Expenditure
- Map visualization displaying geographical distribution of expenditure
- Enables quick regional comparison

### 3. Work Completion Rate by State
- Bar chart comparing efficiency across states
- Highlights top and low-performing regions

### 4. Demand vs Provided Work
- Pie chart showing distribution across years
- Useful for identifying supply-demand gaps

### 5. Person Days Generated
- Table displaying state-wise employment generation
- Helps evaluate labor contribution

---

## 🛠️ Tools & Technologies
- Power BI – Data Visualization  
- DAX – Measures and Calculations  
- Power Query – Data Cleaning & Transformation  

---

## 📂 Dataset
The dataset includes:
- State-wise MGNREGS data  
- Year-wise demand and work provided  
- Expenditure details  
- Person-days generated  

---

## 🚀 Insights
- Demand slightly exceeds supply → indicates employment gap
- Steady increase in expenditure per person day
- High work completion rate (~88%)
- Certain states contribute significantly to person-days generation
