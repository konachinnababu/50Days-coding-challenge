##  Project Overview
This project analyzes the relationship between children's food habits, lifestyle factors, and their overall health status using Excel. The goal is to identify patterns between nutrient intake, screen time, and BMI to support better health decisions.
##  Objectives
* Analyze the impact of fruit and vegetable intake on BMI
* Study the relationship between sugar consumption and screen time
* Compare health indicators across different regions
* Build an interactive dashboard for data-driven insights
##  Tools Used
* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Validation
* Conditional Formatting
## Key Features
### 1. Nutrient Intake vs BMI

* Created PivotTable to analyze average BMI based on daily fruit and vegetable servings
* Used slicer for Region-based filtering

### 2. Sugar & Screen Time Analysis

* Compared average sugar drink consumption and screen time across different health categories
* Visualized using bar chart

### 3. Regional Comparison

* Compared average BMI and fruit intake across regions
* Applied conditional formatting to highlight insights

### 4. Interactive Dashboard

* Built a dynamic dashboard with multiple charts
* Added slicers for Gender and Health Status for real-time filtering

### 5. Data Validation
* Implemented dropdown list for Region to ensure consistent data entry

##  Key Insights
* Higher sugar consumption and screen time are strongly associated with increased BMI
* Children with balanced diets (high fruits & vegetables) tend to have healthier BMI
* Significant regional differences exist in dietary habits and health outcomes

## Recommendations
* Reduce sugary drink consumption among children
* Encourage daily intake of fruits and vegetables
* Promote awareness programs in regions with poor nutrition trends
##  Project Structure
* `Data` – Raw dataset
* `Dashboard` – Interactive charts and analysis
* `Data_Validation` – Dropdown implementation
* `Summary` – Executive summary and insights

## Conclusion

This project demonstrates how Excel can be used as a powerful data analysis and visualization tool to derive meaningful insights and support decision-making in health and nutrition.
