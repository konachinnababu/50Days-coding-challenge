##  Project Overview

This project analyzes logistics and supply chain performance using Power BI.
The goal is to evaluate delivery efficiency, identify delays, and optimize operational costs.

##  Objectives

* Analyze on-time delivery performance
* Identify delayed shipments
* Monitor average delivery time across regions
* Evaluate total delivery cost

##  Dataset Details

The dataset contains shipment-level information with the following fields:

* ShipmentID
* OrderDate
* DeliveryDate
* ExpectedDate
* OriginCity
* DestinationCity
* Region (North, South, East, West)
* Distance_km
* Cost_USD
* DeliveryStatus (On-time, Delayed, Cancelled)
* DeliveryTime_Days

##  Key Performance Indicators (KPIs)

* Total Shipments
* On-time Delivery %
* Average Delivery Time (days)
* Total Delivery Cost



##  DAX Measures Used


##  Key Insights

* Some regions have higher delivery delays
* Average delivery time varies significantly across regions
* Delayed shipments (>2 days) impact overall performance
* Cost increases with distance in many cases

##  Tools Used

* Power BI
* DAX (Data Analysis Expressions)
  
## 📌 Conclusion

This dashboard helps stakeholders monitor delivery performance, reduce delays, and improve supply chain efficiency.
