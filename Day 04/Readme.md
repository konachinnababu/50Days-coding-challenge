##  Objective

Analyze loan portfolio to identify credit risk and profitability trends.

##  Tools Used

* Microsoft Excel
* Pivot Tables
* Charts
* What-If Analysis

##  Key Analysis

* Loan Status vs Funded Amount
* Loan Grade Distribution
* Monthly Lending Trends

##  Advanced Analysis

* Goal Seek for target profit
* Scenario Manager (Optimistic & Pessimistic)
* Data Table for sensitivity analysis

##  Dashboard Features

* KPI metrics
* Interactive slicers
* Dynamic charts

## Insights

* High default rates reduce profitability significantly
* Interest rate changes impact profit directly
* Certain loan grades carry higher risk


## Conclusion

This project demonstrates financial analysis using Excel and helps in strategic decision-making for loan approvals.
